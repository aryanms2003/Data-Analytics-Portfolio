# Bank Loan EDA & Risk Analysis

## 📌 Objective
Identify key drivers of loan default and analyze risk patterns.

---

## 🛠 Tools Used
- Microsoft Excel
- Statistical Functions
- Pivot Analysis

---

## 📊 Analysis Performed
- Missing value treatment
- Outlier detection
- Data imbalance analysis (92% non-default, 8% default)
- Income segmentation analysis
- Occupation-based default trends
- Correlation analysis

---

## 📈 Key Insights
- Data imbalance present (8% default rate)
- Unemployed category showed highest default rate
- Pensioners showed lowest default rate
- Drivers and wait staff had higher risk profiles

---

## 💡 Skills Demonstrated
- Data Cleaning
- IQR Outlier Detection
- Segmented Analysis
- Risk Pattern Identification
