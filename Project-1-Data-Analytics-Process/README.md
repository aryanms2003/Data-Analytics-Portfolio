# Data Analytics Process – Real Life Case Study

## 📌 Objective
Demonstrate the complete Data Analytics Lifecycle using a real-world scenario (PG Hunting).

---

## 🔄 Analytics Lifecycle Applied

1. **Plan** – Define requirements (rent, facilities, safety, accessibility)
2. **Prepare** – Collect data from rental websites and personal research
3. **Process** – Shortlist options based on requirements
4. **Analyze** – Compare shortlisted PGs for optimal decision
5. **Share** – Discuss findings with stakeholders (family/friends)
6. **Act** – Finalize and secure accommodation

---

## 💡 Key Takeaway
This project demonstrates structured problem solving using the end-to-end analytics workflow.

---

## 🛠 Skills Demonstrated
- Problem Structuring
- Analytical Thinking
- Decision Making Framework
- Business-Oriented Approach
