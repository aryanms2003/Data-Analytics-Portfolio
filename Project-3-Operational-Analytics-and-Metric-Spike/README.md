# Operational Analytics & Metric Spike Investigation

## 📌 Objective
Analyze operational datasets to investigate performance metrics and identify anomalies.

---

## 🛠 Tools Used
- MySQL Workbench
- Advanced SQL

---

## 📊 Analysis Performed
- Jobs reviewed per hour (November 2020)
- 7-day rolling average throughput
- Language share percentage analysis
- Duplicate row detection
- Weekly user engagement
- Cohort retention analysis
- Email engagement metrics

---

## 📈 Key Insights
- 7-day rolling average preferred for trend stability
- Persian language had highest share in recent data
- Email open rate: 33.58%
- Email click rate: 14.78%
