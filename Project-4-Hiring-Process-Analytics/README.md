# Hiring Process Analytics (Excel Project)

## 📌 Objective
Analyze hiring data to identify trends in gender distribution, salary ranges, and department structures.

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards

---

## 📊 Analysis Performed
- Gender distribution analysis
- Average salary calculation
- Salary range segmentation
- Department-wise workforce distribution
- Position tier breakdown
- Data cleaning & duplicate removal

---

## 📈 Key Insights
- Male hires: 56.99%
- Average salary offered: 49,925.91
- Operations department had highest workforce
- Position ‘c9’ had highest employee count

---

## 💡 Skills Demonstrated
- Pivot Tables
- Data Cleaning
- Outlier Handling
- Dashboard Creation
