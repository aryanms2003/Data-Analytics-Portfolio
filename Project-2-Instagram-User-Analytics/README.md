# Instagram User Analytics (SQL Project)

## 📌 Objective
Analyze Instagram user interactions using SQL to generate insights on engagement, growth, retention, and bot detection.

---

## 🛠 Tools Used
- MySQL Workbench
- SQL (Joins, Subqueries, Aggregations, CTEs)

---

## 📊 Key Analysis Performed

- Identified 5 oldest users on the platform
- Detected inactive users (26% inactive rate)
- Determined contest winner (highest likes)
- Identified top 5 hashtags
- Found best registration day (Thursday)
- Calculated average posts per user
- Detected potential bot accounts (13 users)

---

## 📈 Key Insights

- 26% users never posted content
- Thursday had highest registrations
- 13 users liked every photo (potential bots)
- Engagement heavily influenced by hashtag usage

---

## 💡 Skills Demonstrated
- GROUP BY
- HAVING
- JOIN
- Subqueries
- Aggregate Functions
- Data-driven business recommendations
