# IMDB Movie Analysis

## 📌 Objective
Identify factors influencing high IMDB ratings and financial performance.

---

## 🛠 Tools Used
- Microsoft Excel
- Statistical Functions
- Data Visualization

---

## 📊 Analysis Performed
- Genre-wise statistical analysis
- Duration vs IMDB score correlation
- Language impact analysis
- Director percentile ranking
- Budget vs Gross earnings correlation

---

## 📈 Key Insights
- Biography genre had highest average rating (excluding single-entry genres)
- Slight positive correlation between duration and rating
- Akira Kurosawa ranked highest in director analysis
- Avatar had highest profit margin

---

## 💡 Skills Demonstrated
- Descriptive Statistics
- Correlation Analysis
- Percentile Ranking
- Business Interpretation
